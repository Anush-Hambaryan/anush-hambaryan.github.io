# Personal Website

Astro site for Anush Hambaryan's personal website and CV PDF.

## Commands

| Command | Action |
| :-- | :-- |
| `npm run dev` | Start the local dev server |
| `npm run build` | Build the production site into `dist/` |
| `npm run preview` | Preview the production build locally |

## Deployment

The site is configured for GitHub Pages through `.github/workflows/deploy.yml`.
After pushing to `main`, enable GitHub Pages with the source set to GitHub Actions.
